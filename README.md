# Quản Lý Nhà Thuốc (Java)

## Giới thiệu
Đồ án nhóm môn Java – xây dựng hệ thống quản lý nhà thuốc.

## Chức năng chính
- Quản lý sản phẩm, danh mục
- Quản lý lô thuốc, hạn sử dụng
- Bán hàng, hóa đơn
- Khuyến mãi, voucher

## Công nghệ sử dụng
- Java
- Swing
- SQL Sever
- JDBC

## Hướng dẫn chạy
1. Import project vào IntelliJ
2. Chạy file pharmacy.sql
3. Run Main.java
